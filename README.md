## Run Locally

Clone the project

```bash
  git clone https://github.com/Sanket4450/Apple.git
```

Go to the project directory

```bash
  cd Apple
```

Install dependencies

#### npm

```bash
  npm install
```

#### yarn

```bash
  yarn
```

#### pnpm

```bash
  pnpm install
```

Run watch command

#### npm

```bash
  npm run watch
```

#### yarn

```bash
  yarn run watch
```

#### pnpm

```bash
  pnpm run watch
```

Run on localhost

ðŸ”— http://localhost:5500/Apple/apple.html
